# Mouse cursor indication

- `--cursor=pointer`
- `--cursor=wait`
- `--cursor=text`
- `--cursor=ew-resize`
- `--cursor=url(https://example.com/cursor.png)`
- and any other standard css cursor value from [CSS#Cursor](https://developer.mozilla.org/en-US/docs/Web/CSS/cursor)

## Fallbacks to

- `--cursor=auto`
