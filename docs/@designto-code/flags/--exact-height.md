# `--exact-height` Flag (Draft)

When applied, this will force dedicated layer's height to be ignore responsive height, use current height as fixed height instead.

## Syntax

```ts
`--exact-height${"="typeof boolean}`
```

## Example

```
--exact-height

--exact-height=true
--exact-height=false

--exact-height=True
--exact-height=False

--exact-height=yes
--exact-height=no
```
