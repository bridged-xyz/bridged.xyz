# Link (Href) data provider

_Examples_

- `--link=https://grida.co`
- `--link=/signup`

## Disclaimer

We choosed name `link` instead of `href` since it is more intuitive across non web developers. Also NextJS has a component named `<Link/>` which suggested us the idea.
